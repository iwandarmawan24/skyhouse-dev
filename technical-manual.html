Role Hierarchy

🔴 superadmin

Akses penuh ke semua fitur termasuk:
• Settings management
• User management
• Contacts & leads
• About Us content
• Semua CRUD CMS

🟡 admin

Akses ke semua konten CMS termasuk:
• Contacts & leads
• About Us
• Semua CRUD konten
✗ Settings
✗ User management

🟢 staff

Akses terbatas ke konten:
• Hero Banners, Products
• Articles, Events
• Facilities, Media
✗ Contacts/Leads
✗ Settings, Users

Notes & Known Items

Dual Contact System

Terdapat dua tabel contact: contacts (legacy, BelongsTo Product) dan contact_submissions (aktif, standalone dengan field residence & project). Frontend Contact form menggunakan contact_submissions. Model Contact kemungkinan tidak lagi digunakan aktif.

Legacy vs Media Library Images

Beberapa model masih memiliki field image (string path legacy) di samping image_uid (FK ke media_library). Controller dan model handles kedua case via fallback logic. Migrasi ke media library sudah dilakukan secara bertahap.

bedrooms Field (Product)

Field bedrooms diubah dari integer ke string (migration 2026-06-17) untuk mendukung nilai seperti "2+1". Form menggunakan text input.

Timezone

Semua datetime menggunakan WIB (Asia/Jakarta). Model Article, Event, MediaHighlight meng-override serializeDate(). Config: config/app.php timezone = Asia/Jakarta.

Route Model Binding

Semua model dengan custom PK (uid) mendefinisikan getRouteKeyName() { return "uid"; } agar route model binding bekerja dengan benar (misalnya MediaLibrary, Career).
